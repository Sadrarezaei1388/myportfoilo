# Ultimate Interactive Portfolio (Next.js 14 + TS + Tailwind + Framer Motion)

A premium, highly‑interactive portfolio starter ready for your personal brand. Inspired by world‑class landing pages with micro‑interactions on every element.

## Quick start

```bash
pnpm i # or npm i / yarn
pnpm dev # http://localhost:3000
```

## Stack
- Next.js 14 (App Router) + TypeScript
- Tailwind CSS
- Framer Motion (scroll progress, reveal, micro‑interactions)
- Lucide icons

Tweak colors in `tailwind.config.ts` and global effects in `app/globals.css`.
Brand name in `components/NavBar.tsx`.
Content in `app/page.tsx`.
